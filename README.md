# 🚀 Escape Room Lógico v2.0
Jogo educativo desenvolvido integralmente em Python, com foco no ensino de conceitos de lógica proposicional de forma interativa, progressiva e gamificada. O projeto simula uma nave interestelar comprometida por um vírus lógico, onde o jogador assume o papel de tripulante para restaurar os sistemas.

Desenvolvido por **Aline de Albuquerque Henriques** para a disciplina de Lógica para Computação (UNICAP).
